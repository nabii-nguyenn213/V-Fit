# V-FIT Constitution

## Rule

Use `.specify/memory/constitution.md` as the authoritative Spec Kit
constitution.

## Current Version

Version 1.0.0, ratified on 2026-05-28.

## Mandatory Principles

- Treat AI as the native control plane.
- Store prompts under `skills/conversation/`.
- Store architecture decisions under `skills/tech-decision/decision.md`.
- Validate every AI input before model use.
- Parse and constrain every AI output before app use.
- Gate premium AI by onboarding, subscription, JWT, rate limits, and fallback.
